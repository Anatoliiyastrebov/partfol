# Папка для изображений

Загрузите сюда ваши фотографии:

## Рекомендуемые файлы:

1. **photo.jpg** или **photo.png** - ваше основное фото (для секции Hero)
   - Рекомендуемый размер: 400x400px или больше
   - Формат: JPG или PNG
   - Оптимизируйте изображение для веба (можно использовать онлайн-инструменты)

2. **about-photo.jpg** - фото для секции "О себе" (опционально)

3. **project-1.jpg**, **project-2.jpg**, **project-3.jpg** - изображения для проектов

## Как оптимизировать изображения:

- Используйте инструменты типа TinyPNG, Squoosh или ImageOptim
- Рекомендуемый размер для фото в Hero: 400-600px по ширине
- Формат WebP предпочтительнее для лучшей производительности

## После загрузки:

1. Переименуйте ваше фото в `photo.jpg` или `photo.png`
2. Обновите страницу в браузере
3. Фото автоматически появится в портфолио
