# Инструкция по настройке отправки сообщений на Gmail

Эта инструкция поможет вам настроить форму обратной связи для отправки сообщений на ваш Gmail через EmailJS.

## Шаг 1: Регистрация на EmailJS

1. Перейдите на сайт [EmailJS.com](https://www.emailjs.com/)
2. Нажмите **"Sign Up"** (Регистрация)
3. Зарегистрируйтесь через Google аккаунт (рекомендуется) или email
4. Подтвердите регистрацию через email

## Шаг 2: Подключение Gmail

1. Войдите в аккаунт EmailJS
2. Перейдите в раздел **"Email Services"** (Email Сервисы)
3. Нажмите **"Add New Service"** (Добавить новый сервис)
4. Выберите **"Gmail"**
5. Нажмите **"Connect Account"** (Подключить аккаунт)
6. Войдите в свой Gmail аккаунт и разрешите доступ
7. После подключения вы увидите **Service ID** (например: `service_xxxxxxx`)
   - **Скопируйте этот ID** - он понадобится позже

## Шаг 3: Создание шаблона письма

1. Перейдите в раздел **"Email Templates"** (Шаблоны писем)
2. Нажмите **"Create New Template"** (Создать новый шаблон)
3. Заполните форму:

   **Subject (Тема письма):**
   ```
   Новое сообщение с сайта портфолио от {{from_name}}
   ```

   **Content (Содержание письма):**
   ```
   Вы получили новое сообщение через форму обратной связи на вашем сайте.
   
   Имя отправителя: {{from_name}}
   Email отправителя: {{from_email}}
   
   Сообщение:
   {{message}}
   ```

4. В поле **"To Email"** (Кому) укажите **ваш Gmail адрес** (например: `your.email@gmail.com`)
5. В поле **"From Name"** (От имени) можно указать: `Портфолио сайт`
6. Нажмите **"Save"** (Сохранить)
7. После сохранения вы увидите **Template ID** (например: `template_xxxxxxx`)
   - **Скопируйте этот ID** - он понадобится позже

## Шаг 4: Получение Public Key

1. Перейдите в раздел **"Account"** → **"General"** (Аккаунт → Общие)
2. Найдите раздел **"API Keys"** (API Ключи)
3. Скопируйте **"Public Key"** (например: `xxxxxxxxxxxxxxxx`)
   - **Скопируйте этот ключ** - он понадобится позже

## Шаг 5: Настройка config.js

1. Откройте файл `config.js` в вашем проекте
2. Замените значения на те, что вы скопировали:

```javascript
const CONFIG = {
    email: {
        enabled: true,  // Включаем отправку на email
        serviceId: 'service_xxxxxxx',      // Service ID из шага 2
        templateId: 'template_xxxxxxx',    // Template ID из шага 3
        publicKey: 'xxxxxxxxxxxxxxxx',     // Public Key из шага 4
        toEmail: 'your.email@gmail.com'    // Ваш Gmail адрес
    },
    useMethod: 'email'  // Используем только email
};
```

3. Сохраните файл

## Шаг 6: Проверка работы

1. Откройте ваш сайт в браузере
2. Перейдите к форме обратной связи
3. Заполните форму и отправьте тестовое сообщение
4. Проверьте ваш Gmail - сообщение должно прийти в течение нескольких секунд

## Важные замечания

- **Бесплатный план EmailJS** позволяет отправлять до 200 писем в месяц
- Письма могут попасть в папку **"Спам"** - проверьте её тоже
- Если письма не приходят, проверьте:
  - Правильность Service ID, Template ID и Public Key
  - Что Gmail аккаунт подключен в EmailJS
  - Консоль браузера на наличие ошибок (F12 → Console)

## Решение проблем

### Письма не приходят
1. Проверьте папку "Спам" в Gmail
2. Убедитесь, что все ID и ключи введены правильно
3. Проверьте консоль браузера (F12) на наличие ошибок
4. Убедитесь, что `enabled: true` в config.js

### Ошибка "Service not found"
- Проверьте правильность Service ID
- Убедитесь, что Gmail сервис подключен в EmailJS

### Ошибка "Template not found"
- Проверьте правильность Template ID
- Убедитесь, что шаблон сохранен в EmailJS

### Ошибка "Invalid API key"
- Проверьте правильность Public Key
- Убедитесь, что используете Public Key, а не Private Key
